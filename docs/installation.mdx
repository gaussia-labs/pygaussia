## Requirements

- Python **≥ 3.11**
- Package manager: `pip`, `uv`, or `poetry`

## Base installation

The base package includes the core framework, schemas, and metrics that only require an LLM (Context, Conversational, BestOf):

<CodeGroup>

```bash pip
pip install gaussia
```

```bash uv
uv add gaussia
```

```bash poetry
poetry add gaussia
```

</CodeGroup>

## Optional extras

Some metrics require additional dependencies for ML models, embeddings, or statistical analysis. Install only what you need:

<CodeGroup>

```bash Toxicity
pip install "gaussia[toxicity]"
```

```bash Bias
pip install "gaussia[bias]"
```

```bash Vision
pip install "gaussia[vision]"
```

```bash Humanity
pip install "gaussia[humanity]"
```

```bash Regulatory
pip install "gaussia[regulatory]"
```

```bash Explainability
pip install "gaussia[explainability]"
```

```bash All metrics
pip install "gaussia[all]"
```

</CodeGroup>

### What each extra includes

| Extra | Additional dependencies | Used by |
|---|---|---|
| `toxicity` | nltk, numpy, pandas, scikit-learn, sentence-transformers, hdbscan, umap-learn, torch | Toxicity metric |
| `bias` | torch | Bias metric |
| `vision` | numpy, sentence-transformers, torch | VisionSimilarity, VisionHallucination |
| `humanity` | numpy, pandas | Humanity metric |
| `regulatory` | torch, accelerate | Regulatory metric |
| `explainability` | interpreto, torch, transformers | Token attribution analysis |
| `generators` | *(no extra deps)* | Synthetic dataset generation |
| `prompt-optimizer` | optuna | Prompt optimization |
| `metrics` | All metric extras combined | All metrics |
| `all` | Everything above | Full installation |

## Verify installation

```python
import gaussia
print(gaussia.__version__)
```

<Tip>
  For development, we recommend using [uv](https://docs.astral.sh/uv/) for fast dependency resolution: `uv add "gaussia[all]"`.
</Tip>
