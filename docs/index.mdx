## What is gaussia?

Gaussia is an AI evaluation framework that provides metrics for measuring fairness, toxicity, bias, conversational quality, and more for AI models and assistants. The Python SDK (`gaussia`) is the reference implementation.

## Available metrics

Gaussia provides a comprehensive suite of evaluation metrics across multiple dimensions:

| Metric | What it measures | Requires LLM |
|---|---|---|
| [Context](/metrics/context) | Response alignment with provided context | ✅ |
| [Conversational](/metrics/conversational) | Dialogue quality via Grice's maxims | ✅ |
| [Toxicity](/metrics/toxicity) | Toxic language and demographic bias (DIDT) | ❌ |
| [Bias](/metrics/bias) | Bias across protected attributes | ✅ |
| [Humanity](/metrics/humanity) | Emotional profile and entropy | ❌ |
| [BestOf](/metrics/best-of) | Tournament-style assistant comparison | ✅ |
| [Agentic](/metrics/agentic) | Agent correctness with pass@K | ✅ |
| [Vision](/metrics/vision) | VLM hallucination and similarity | ❌ |
| [Regulatory](/metrics/regulatory) | Compliance against regulatory corpus | ❌ |
